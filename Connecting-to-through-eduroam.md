# Connecting to your Pi over Eduroam

Replace **_[00]_** with your interaction engine number.
For SSH use:
```
ssh pi@ct-devicefarm-1257.tech.cornell.edu -p 122[00]  
```

Connect to the Website
```
ct-devicefarm-1257.tech.cornell.edu:280[00]
```
